# Templates and Checklist Templates

## What to build

Add editable Wedding Templates and Checklist Templates so common Indian/Hindu wedding planning structures can be created quickly and then customized. Templates should create editable Events, Tasks, Vendor Categories, Budget Items, Rituals, Planning Areas, and Items. Checklist Templates should apply reusable Task sets to Events, Vendors, Rituals, Planning Areas, and Items.

## Acceptance criteria

- [ ] A Planner or Owner can apply an editable Wedding Template to the single Wedding.
- [ ] Templates can create common Events such as Haldi, Mehndi, Sangeet, Ceremony, and Reception.
- [ ] Templates can create starter Tasks, Vendor Categories, Budget Items, Rituals, Planning Areas, and Items.
- [ ] Created records are editable after template application.
- [ ] Checklist Templates can be applied to supported records to create reusable Task sets.
- [ ] Tests cover applying templates, editing created records, and applying checklist templates.

## Blocked by

- Event and Task Planning Loop
- Vendor Pipeline
- Budget and Payment Tracker
- Rituals and Items
