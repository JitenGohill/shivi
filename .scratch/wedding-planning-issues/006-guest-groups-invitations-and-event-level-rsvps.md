# Guest Groups, Invitations, and Event-Level RSVPs

## What to build

Add guest planning records without guest platform access. Planners should be able to create Guests, Guest Groups, Guest Segments, Invitations, and manual RSVPs that vary by Event. The Dashboard should show invitation and RSVP snapshots.

## Acceptance criteria

- [ ] A Planner or Owner can create Guests as planning records only.
- [ ] Guests can be grouped into Guest Groups for household, family, or shared invitation workflows.
- [ ] Guests can be labeled with flexible Guest Segments.
- [ ] Invitations track Not Sent, Ready, Sent, Responded, and Follow-up Needed states without sending messages.
- [ ] RSVPs are manually recorded by Collaborators.
- [ ] RSVP state can differ per Event for the same Guest or Guest Group.
- [ ] The Dashboard shows invitation status counts and RSVP counts by Event.
- [ ] Tests prove event-level RSVPs do not collapse into a single Wedding-level answer.
