# Shivi

A private single-Wedding planning workspace for a multi-event Indian/Hindu wedding.

## Local Development

Install dependencies and start the app:

```bash
npm install
npm run dev
```

Then open `http://127.0.0.1:3000`.

## Tests

```bash
npm run test:smoke
```

The smoke check runs a production Next build, which validates that the Dashboard
and placeholder planning routes render for the single seeded Wedding.
